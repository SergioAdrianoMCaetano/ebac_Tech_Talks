Front-end

O front-end é a parte de um sistema ou aplicativo que os usuários interagem diretamente. Envolve a interface gráfica, como botões, menus e layouts, e a experiência do usuário ao utilizar o produto. Desenvolvedores front-end usam tecnologias como HTML, CSS e JavaScript para criar interfaces responsivas e interativas.

UX | UI

UX (User Experience) e UI (User Interface) são disciplinas relacionadas à experiência do usuário em um produto digital. UI refere-se à interface gráfica, ou seja, ao design visual e à disposição de elementos na tela. UX engloba a experiência geral do usuário, incluindo a usabilidade, a acessibilidade e a satisfação ao usar o produto. Ambas são essenciais para criar produtos digitais bem-sucedidos

Data Science

Data Science é uma disciplina que utiliza métodos científicos, processos, algoritmos e sistemas para extrair conhecimento e insights de dados em diversas formas. Envolve a análise, interpretação e apresentação de dados para entender padrões, tendências e informações relevantes para apoiar a tomada de decisões. Os profissionais de Data Science geralmente têm habilidades em programação, estatística, machine learning e áreas relacionadas para explorar e analisar grandes volumes de dados de maneira significativa e útil.

Back-end

O back-end é a parte de um sistema que lida com a lógica, processamento de dados e interações com o servidor. Ele não é visível para o usuário final, mas é fundamental para o funcionamento de um aplicativo. Desenvolvedores back-end trabalham com tecnologias como servidores, bancos de dados e linguagens de programação para criar a lógica e a estrutura do sistema.
